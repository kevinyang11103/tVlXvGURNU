# Python医疗设备管理系统 python734

## 项目概述

本项目是基于Python语言的医疗设备管理系统，集成了源码、运行文档及详尽的论文。系统采用成熟的技术栈，致力于为医疗机构提供高效、稳定的设备管理解决方案。

## 技术栈

- 后端：Django
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript

## 功能模块

- **用户模块**
  - 登录注册
  - 个人信息修改
- **采购管理**
  - 采购审批
- **供应商管理**
  - 供应商信息管理
- **设备管理**
  - 设备信息管理
  - 库存管理
  - 设置库存和设备名称

## 系统角色

- 系统管理员
- 普通用户
- 采购人员
- 供应商

- Python 3.x

- 支持主流浏览器
